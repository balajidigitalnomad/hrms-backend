# HRMS Backend

Enterprise Human Resource Management System developed using Java 17 and Spring Boot 3.

## Features
- Employee Management
- Department Management
- Designation Management
- User & Role Management
- REST APIs
- Bean Validation
- Global Exception Handling
- Layered Architecture

## Technology Stack
- Java 17
- Spring Boot 3
- Spring Data JPA
- Hibernate
- MySQL
- MapStruct
- Maven
- Git

## Project Architecture

Controller
   ↓
Service
   ↓
Repository
   ↓
Database
